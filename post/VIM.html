安装vim

CentOS

​ sudo yum -y install vim

Ubuntu

​ sudo apt-get install vim

Arch

​ sudo pacman -S vim

打开VIM

  vim或vim [file] //把[file]替换成你想打开的文件

  例子：

  vim c.txt //用vim打开名叫c的txt文件

VIM命令(推荐自己尝试一下)

  VIM中的模式

      VIM中有三种比较常用的模式：

           NORMAL MODE // 普通模式 也是默认打开的模式 大部分命令都是在普通模式下使用的

           INSERT MODE // 写入模式

           VISUAL MODE // 可视模式

​

  普通模式中的命令

        操作 + 动作 (operation + motion)</u>

           例子 dh d 操作(delete删除) h 动作(左) 连起来就是“向左删除一个字符”

        命令可以连用(带冒号的) 也可以带一个!表示强制…

           例子 :wq保存(:w)并退出(:q)

           :q! (quit!)强制退出vim

        ESC键 进入默认模式

     进入插入模式

        i键 (insert)在文字的前方进入写入模式

        I键 (大写i)在句子的开头插入

        a键 (append)在文字的后方进入写入模式

        A键 (大写A)在句子的末尾插入

        o键 (open a new line)在当前行的下方开启一个新行并进入插入模式

        O键 (大写o)在当前行的上方开启一个新行并进入插入模式

     命令

        :w (write)保存文件

        :q (quit)退出vim

       :e [路径] //把[路径]换成你自己的路径 打开文件

     操作

        d (delete)实际上是剪切

           dd剪切一行

           d + 数字 + 方向 剪切[你的方向]的[你的数字]个字符

              例子

           d3h 向左删除三个字符

           dh 向左删除一个字符

        p (paste)粘贴你刚刚复制的字符

        y (yank)复制

           例子

              y3h 向左复制三个字符

              yh 向左复制一个字符

        c (change)删除完成后进入写入操作(和d操作相同)

        0 回到这行的开头

        u 撤销

        n 下一条搜索 N 上一条搜索

       gg 移到开头

       G 移到最后一行

​

     动作

        h左 j下 k上 l左 移动光标*（可以按照自己的喜好更改 后面再说）*

           如果要一次性移动多行可以输入 [你要多少行] + [你移动光标方向按键]

           这个也是前面说的操作 + 动作

              例子 10k 向上(k)移动十行(10)

        f (find)查找 f + 你要查找的字母

           fv 查找字母v

  可视模式(命令和普通模式相同)

       在普通模式下按下v键进入可视模式(普通可视)

       在普通模式下按下V键(大写v)进入可视模式(行可视)

       在普通模式下按下ctrl+v进入可视模式(块可视)

    normal指令

       在可视模式下输入: normal + 你在默认模式下的命令 就会在可视模式下执行(v/ v-line/ v-block都可以 我懒得写其他的例子了 自己尝试吧)

         例子 :normal Idef-md //把选中行(v-line行可视)的前方插入(I)def-md(中间不能有空格 要不然会把空格一块加上)

按键重映射

        .vimrc是vim的配置文件 后面的配置都是放在这个文件下

   打开.vimrc

        mkdir .vim //创建.vim文件夹.vimrc要放在这个文件夹下

        cd .vim //进入.vim文件夹

        vim ~/.vimrc //如果没有这个文件的话 vim会打开时自动创建

  编辑.vimrc

        注意: 你自己改完键之后上面的操作/ 动作会变成你自己绑的键
